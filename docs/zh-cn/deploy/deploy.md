# 安装部署

## Window操作系统

* 安装Java环境
* 下载[安装包](https://gitee.com/648823596/dnsserver/attach_files/707070/download/DNSServer-1.0.zip)
* 下载压缩文件后解压,修改里面config.properties文件内容,指定mysql数据库地址,用户名和密码等信息
* 双击start.bat即可
* 访问http://127.0.0.1:9999
